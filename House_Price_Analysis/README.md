# House Price Analysis

## Objective
Perform Exploratory Data Analysis (EDA) on a house price dataset.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- VS Code

## Files
- house_price.csv
- cleaned_dataset.csv
- eda_report.ipynb

## Steps Performed
1. Data Loading
2. Data Cleaning
3. Missing Value Treatment
4. Duplicate Removal
5. Statistical Analysis
6. Data Visualization
7. Correlation Analysis

## Conclusion
House price is strongly influenced by area and number of bedrooms.
